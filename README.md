# 編み物アプリ - 使い方

## アプリの起動方法

1. `index.html` をブラウザで開く
2. 検索ボックスに編み目の名前を入力
3. 検索ボタンを押す

## YouTube動画の追加方法

埋め込みコードをそのまま使う方式なので、とても簡単です!

### 手順1: YouTube埋め込みコードの取得

1. YouTubeで使いたい動画を開く
2. 動画の下にある「**共有**」ボタンをクリック
3. 「**埋め込む**」を選択
4. 表示されたコード全体をコピー
   ```html
   <iframe width="560" height="315" src="https://www.youtube.com/embed/XXXXX...
   ```

### 手順2: script.jsに貼り付け

`script.js` を開いて、以下の部分に埋め込みコードを貼り付け:

```javascript
{
    name: "細編み",
    description: "...",
    embedCode: ""  // ← ここに埋め込みコードを貼り付け
}
```

例:
```javascript
{
    name: "細編み",
    description: "...",
    embedCode: '<iframe width="560" height="315" src="https://www.youtube.com/embed/ABC123..." allowfullscreen></iframe>'
}
```

**注意:** 埋め込みコードは必ず `'` (シングルクォート) か `"` (ダブルクォート) で囲んでください!

### おすすめの検索キーワード

- 中長編み かぎ針 編み方

- 長編み かぎ針 基本

## 編み目データの追加方法

`script.js` の `stitchDatabase` 配列に新しいオブジェクトを追加:

```javascript
{
    name: "新しい編み目",
    description: "説明文",
    embedCode: '<iframe width="560"...(埋め込みコード全体)...</iframe>'
}
```

## トラブルシューティング

### 動画が表示されない
- 埋め込みコードが正しく貼り付けられているか確認
- 埋め込みコードが `'` または `"` で囲まれているか確認
- YouTubeの「共有」→「埋め込む」から取得したコードか確認
- インターネットに接続されているか確認

### 検索しても結果が出ない
- 入力した編み目名がデータベースにあるか確認
- script.jsのstitchDatabaseを確認
